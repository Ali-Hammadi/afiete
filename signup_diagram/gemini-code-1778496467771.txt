================================= [ PHASE 4: SESSION HANDLING (DIO) ] ===========================
(Happens automatically in the background for ANY authorized request if the access token expires)

       [ Any Request via Dio ]
                 │
                 V
     [ HTTP 401 UNAUTHORIZED ] ─────────┐
                 ^                      │
                 │                      V
      [ Retry Original Request ]  [ Dio Error Interceptor ]
                 ^                      │
                 │                      │ 1. Lock Interceptor (dio.interceptors.requestLock)
                 │                      │ 2. Read refresh_token from Secure Storage
                 │                      │ 3. POST /api/auth/refresh
                 │                      │
                 └──────(Success)───────┤ -> Save New access_token & refresh_token
                                        │
                                        │
                         (Failed)───────┤ -> Emit SessionExpired -> Clear Storage
                                        │ -> GoRouter redirects to '/signup'
